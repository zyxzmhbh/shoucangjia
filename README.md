# 收藏夹（SillyTavern 扩展）

用于在酒馆聊天中收藏高亮句子，并保存可回看的上下文快照。

## 已实现

- 扩展页入口名称：`收藏夹`
- 在扩展页可直接点击：
  - `打开收藏夹`
  - `收藏选中`
  - `高亮并收藏`
- 手机端长按选中文本后，页面底部会出现 `收藏选中` 按钮
- 收藏时可填备注、标签
- 收藏夹支持：
  - 角色筛选
  - 会话筛选
  - 备注筛选
  - 标签筛选
  - 全文搜索
  - 跳转原消息（同会话）
- 快照保存：
  - 角色卡信息
  - 上下各 5 条聊天记录
  - 收藏时间、备注、标签、会话标识

## 安装

在 SillyTavern 的扩展安装里填：

```text
https://github.com/zyxzmhbh/shoucangjia.git
```

## 手机端使用说明

1. 在聊天区长按文本进行系统选中。
2. 不需要关闭系统复制菜单。
3. 页面底部出现 `收藏选中` 后点击即可保存。
4. 去扩展页 `收藏夹` 打开面板查看。
