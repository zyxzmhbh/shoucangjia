# 收藏夹（SillyTavern 扩展）

## 当前交互

- 扩展页中显示 `收藏夹`，与常见扩展同样的折叠样式（单行标题，右侧展开按钮）。
- 展开后直接显示收藏夹内容（筛选、收藏列表、导入导出）。
- 在聊天中选中文本后弹出：
  - `收藏`
  - `高亮并收藏`
  - `取消`
- 手机端弹条固定在底部并提高层级，尽量避免被系统复制菜单遮挡。

## 数据

- 收藏文本、备注、标签
- 角色信息快照
- 上下各 5 条聊天记录
- 会话标识（区分同角色不同会话）
