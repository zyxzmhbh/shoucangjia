# 收藏夹（SillyTavern 扩展）

用于在酒馆聊天中收藏句子并保存上下文快照。

## 交互方式（当前）

- 扩展页里显示 `收藏夹`，和其他扩展一样可展开/收起。
- 展开后直接就是收藏夹内容（筛选、列表、导入导出）。
- 在聊天区选中文本后会弹出操作条：
  - `收藏`
  - `高亮并收藏`
  - `取消`
- 手机端也会显示该操作条（固定在底部，避免被系统菜单遮挡）。

## 收藏内容

- 选中文本
- 收藏时间
- 备注、标签
- 角色卡快照
- 上下各 5 条聊天记录
- 会话标识（区分同角色不同会话）

## 安装

```text
https://github.com/zyxzmhbh/shoucangjia.git
```

## 手机端建议

1. 长按消息文本完成系统选中。
2. 看到底部收藏操作条后，直接点 `收藏` 或 `高亮并收藏`。
3. 去扩展页展开 `收藏夹` 查看结果。
